Sie müssen Ihren Code nur in einen Funktionsausdruck einpacken und das Ergebnis oder den gesamten Ausdruck zurückgeben. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

function sumUptoN(N: number): number {

    return N * (N + 1) / 2;
}
