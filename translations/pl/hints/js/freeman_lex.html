Proszę złapać pomysł 💡! Rozwiązanie, które Państwu proponuję jest niezwykle proste. Proszę poczytać o progresji arytmetycznej i jej użyć!

Ok, od czego powinienem zacząć?

Proszę poczytać o arithmetic progression, sumie jego elementów i wykorzystać te informacje! 

S = N * (N + 1) / 2;
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne i zwrócić wynik lub całe wyrażenie. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Poniżej znajduje się całe rozwiązanie:

function sumUptoN(N: number): number {

    return N * (N + 1) / 2;
}

Jasne, proszę pobrać!

To rozwiązanie wykorzystuje operator trójargumentowy i rekurencję. 

function sumUptoN(N: number): number {

    return N == 1? 1: N + sumUptoN(N-1);
}
