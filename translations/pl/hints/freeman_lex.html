Proszę złapać pomysł 💡! Rozwiązanie, które Państwu proponuję jest niezwykle proste. Proszę poczytać o progresji arytmetycznej i z niej skorzystać!

Ok, od czego powinienem zacząć?

Proszę poczytać o arithmetic progression, sumie jego elementów i wykorzystać te informacje! 

S = N * (N + 1) // 2
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne i zwrócić wynik lub całe wyrażenie. Zaleca się używanie podpowiedzi typu dla danych wejściowych i wyjściowych.

Oto całe rozwiązanie:

def sum_upto_n(N: int) -> int:

    return N * (N + 1) // 2

Jasne, proszę łapać!

Proszę użyć funkcji range(). 

def sum_upto_n(N: int) -> int:

    return sum(range(1, N + 1))

To rozwiązanie wykorzystuje rekurencję. 

def sum_upto_n(N: int) -> int:

    return 1 if N == 1 else N + sum_upto_n(N-1)
